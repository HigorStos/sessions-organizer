This is a React app powered by Vite and Tailwind CSS.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the main UI by modifying [src/App.tsx](src/App.tsx). The page auto-updates as you edit the file.

## Learn More

To learn more about Vite and React, take a look at these resources:

- [Vite Documentation](https://vite.dev/guide/)
- [React Documentation](https://react.dev/learn)

## Deploy

Build with `npm run build` and serve the generated `dist` folder with any static host.
